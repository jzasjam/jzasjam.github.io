Raspberry Pi & SenseHAT Projects

Pi SenseHAT LED Designer

  • Project Details View on GitHub

Project Description

  • Node.js application which allows user to design LED colour configuration and output this live to the SenseHAT as well as providing the LED matrix values.
  • Also includes a simple tool to display a scrolling custom message in chosen colours on the SenseHAT LEDs

Visualise Pi SenseHAT Sensor Data Sent to Azure IoTHub

  • Project Details
    Part 1 on GitHub Part 2 on GitHub

Project Description

  • Part 1 - Send Data To Azure IoT Hub From the Raspberry Pi based on tutorial: https://github.com/Azure-Samples/azure-iot-samples-node/tree/master/iot-hub/Tutorials/RaspberryPiApp
  • Part 2 - NodeJS Data Visualisation for Azure IotHub adapted from https://github.com/Azure-Samples/web-apps-node-iot-hub-data-visualization with Google Charts

  • The IMU (Innertial Measurement Unit) data from the SenseHAT is also sent to the Azure IoT Hub, visualised in a similar way and rendered in 3D using Three.js
